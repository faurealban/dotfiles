## Installation
If you only want some specific configuration, search in this repo.\
If you want everything: run `install.zsh`

> [!WARNING]
> This will overwrite your configs. Backup recommended. Use at your own risk.

After istallation, do:
```
nvim
:Lazy sync
:MasonInstallAll
```
